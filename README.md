## 🟣 About Me

```yaml
role: "Cybersecurity Engineering Student"
focus: "SOC Operations · Blue Team · Detection Engineering"
current_build: "RADAR — Real-time Autonomous Defense And Response"
```

I'm a Cybersecurity Engineering student who builds and *operates* SIEM environments end‑to‑end — from raw log pipeline ingestion to tuning detection rules that separate genuine attacker signal from background noise. I care less about collecting alerts and more about making the *right* alert fire, fast, with context an analyst can act on.

My flagship project, **RADAR**, brings this philosophy together in a live network monitoring platform that captures traffic in real time, maps every detection to **MITRE ATT&CK**, and uses AI to auto-generate incident containment playbooks — cutting down manual analyst effort. I've paired this hands-on lab work with structured practice, including phishing-alert triage through **TryHackMe's SOC Simulator**.
