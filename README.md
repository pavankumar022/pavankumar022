## 🚀 Featured Project

### [RADAR — Real-time Autonomous Defense And Response](https://github.com/pavankumar022/RADAR)
`FastAPI` `React` `WebSockets` `tcpdump` `MITRE ATT&CK` `Gemini API`

A live 3D WebGL threat globe visualizing real-time attacker geolocation, with AI-generated incident containment playbooks and analyst-ready PDF reports. Captures live packets across multi-port sensors (SSH, RDP, SMB, DB) to detect port scans, brute-force attempts, and C2-beacon activity — mapped to a live MITRE ATT&CK coverage heatmap and stress-tested at up to 500 events/sec.
